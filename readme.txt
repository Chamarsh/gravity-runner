Description- Gravity Run. Player will control a character and change gravity at will to try to dodge the 
objects, collect coins and powerups and progress throughout the level. If the player gets stopped and ends up 
going off the screen then the player loses the game

How to run- load the pictures and the cmu 112 graphics module into the same directory as the project code. 
This should result in one being able to load and play the game

Libraries- cmu graphics 112 and random are the only libraries loaded but they are preinstalled so
there are no outside libraries needed
